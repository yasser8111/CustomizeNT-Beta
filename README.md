# CustomizeNT (Beta)

![Zen Minimalist Preview 1](previwe-1.png)
![Zen Minimalist Preview 2](previwe-2.png)

CustomizeNT is a professional, minimalist, and highly customizable browser "New Tab" extension. It provides a zen-like experience for managing your favorite links with elegant design and smooth performance.

## ✨ Features

- **Zen Minimalism**: A clean, distraction-free interface that puts your content first.
- **Dynamic Organization**: Group your bookmarks into cards and arrange them with drag-and-drop.
- **Professional Customization**:
  - Adjust column counts (2-12).
  - Scale cards (60% - 150%) to fit any screen size.
  - Custom primary colors and theme modes (Light/Dark).
- **Interface Modes**: 
  - **Standard**: Full titles and guides.
  - **Simple Mode**: Icon-only navigation for the ultimate minimalist look.
- **Glassmorphism UI**: Beautiful semi-transparent surfaces that blend perfectly with your background.
- **Background Support**: Use preset images or upload your own photos and videos.
- **RTL Support**: Full native support for Arabic and RTL languages.

## 🚀 Getting Started

1. Download the repository.
2. Open Chrome/Edge and navigate to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked** and select the project folder.

## 🛠️ Technology Stack

- **Pure HTML5/CSS3**: No heavy frameworks, ensuring instant load times.
- **Vanilla JavaScript**: Optimized performance with an MVC-inspired architecture.
- **SortableJS**: Smooth, native-feel drag and drop.
- **Lucide Icons**: Clean, consistent iconography.

## 🎨 Interface Controls

- **Folder Plus**: Create a new group.
- **Book Marked**: Choose from curated templates.
- **Palette**: Customize colors, themes, and backgrounds.
- **Settings**: Adjust grid layout, card sizes, and interface modes.
